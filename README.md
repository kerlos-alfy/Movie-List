# 🎬 **Movie List**

## 🌟 **Introduction**

Welcome to the **Movie List** – an interactive web application where movie lovers can explore trending and popular movies! By integrating data from the **[TMDb API](https://www.themoviedb.org/)**, this app allows users to **search**, **discover**, and **dive into** detailed movie information, including ratings, reviews, and more. Whether you're a film buff or just looking for your next binge-worthy movie, this app has you covered!

---

## ⚡ **Key Features**

✨ **Discover Popular Movies**: View the latest and trending movies with just a click!  
🔍 **Search Movies by Keyword**: Find your favorite films easily by searching with keywords.  
🎬 **Detailed Movie Information**: Get all the essential details like title, release date, ratings, and reviews.  
🌀 **Seamless Pagination**: Navigate between pages of movie results with ease.

---

## 🛠️ **Technologies Used**

- **React**: Building an interactive and dynamic user interface.
- **React Bootstrap**: Responsive, mobile-first UI components.
- **Axios**: Fetching movie data from the TMDb API.
- **React Router**: Smooth navigation between different pages.
- **React Paginate**: Implementing an easy-to-use pagination system.

---

## 🚀 **How to Use**

### 1. **Clone the Repository**
   Clone this project to your machine with:
   ```bash
   git clone https://github.com/kerlos-alfy/Movie-List.git
   ```

### 2. **Install Dependencies**
   Once cloned, navigate into the project folder and install the necessary dependencies:
   ```bash
   npm install
   ```

### 3. **Run the Application**
   To start the app locally, run:
   ```bash
   npm start
   ```
   Your app will be live at `http://localhost:5173`.

---

## 🔧 **Interacting with the Project**

1. **Search for Movies**: Use the search bar at the top to find your favorite films!  
2. **View Movie Details**: Click on any movie to see its detailed information, including plot, ratings, and more.  
3. **Navigate Pages**: Use the pagination controls at the bottom to easily browse through different pages of movie results.
